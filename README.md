# opencode-buddy

A fun, gamified companion plugin for OpenCode! Your Buddy lives in your terminal, reacts to your code executions, and levels up as you work.

## Features (Planned)
- Hatches a unique companion based on your local identifier.
- Buddy gains XP with every execution.
- LLM-powered personality module to make your buddy chat and react.
- ASCII sprites and dynamic moods.
